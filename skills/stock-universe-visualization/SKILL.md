Be concrete (call out patterns, regimes, concentrations, outliers).

Use trend following concepts previously provided: ModeBucket (Dark/Light Green/Red), MoM, RevL distance, duration (Days/Weeks), P/L.

Produce actionable outputs (lists of tickers and what to do with them), not vague commentary.

Universes + Files (mental model)

I use two categories:

Core Universe = “Stock Universe Dashboard”
Big “lay of the land”: where the whole universe sits by trend regime and duration, with P/L and RevL proximity.

Strategic Universes
Purpose-built slices that surface tradeable situations:

Top Momentum Movers (MoM 7–10)

Daily Buy Strategic Universe (names in daily buy mode; day-bins)

Switch Universe (Daily Dark Green + Weekly Light Red = “down weekly but ripping daily” setup)

TrendLen/Momentum scatter (duration vs momentum with RS “R” color/labels)

Required reading of variables

Interpret these consistently:

ModeBucket logic

Close ≥ RevL = bull regime; Close < RevL = bear regime

MoM > 0 = rising momentum; MoM < 0 = falling momentum

So:

Dark Green = bull + rising MoM (strong trend)

Light Green = bull + falling MoM (bull but momentum fading/consolidation)

Light Red = bear + rising MoM (bear trend, but momentum turning up = potential reversal setup)

Dark Red = bear + falling MoM (weak / breakdown regime)

Duration

“Time in Trend / Days / Weeks” = how long in that regime. Longer duration = more “crowded/extended” (or persistent weakness).

P/L

P/L is the RL table P/L %. Use it as a “who’s been rewarded/punished” score over the RL period.

Distance to RevL / QuickTargets

Distance to RevL: how stretched vs regime line.

QuickTargets: intermediate “snapback” level surfaced from RL hover; treat as tactical reference not a thesis.

R field

“R” is Relative Strength indicator used in some visuals. Higher = stronger vs peers (or whatever RL defines). Use it for ranking within a bin.

How to analyze each visualization
1) Core Universe (Stock Universe Dashboard)

You’ll see scatter(s) with ModeBucket colors, duration on one axis, P/L on another, plus “shadow” dots (prior week) sometimes.

Do this:

Regime mix: summarize counts/percentages in each ModeBucket and what that implies (risk-on vs risk-off, breadth).

Rotation: identify if Light Red count is rising (early reversal attempts) or Dark Red is rising (weakening).

Crowding: point out long-duration Dark Green with high P/L (late-stage winners) vs long-duration Dark Red with very negative P/L (capitulation candidates).

Proximity opportunities: call out names within ~5% of RevL (especially Light Red near RevL = “reclaim candidates”; Light Green near RevL = “don’t lose it” candidates).

Movement vs prior week (shadow dots): identify biggest regime flips (e.g., Dark Red → Light Red; Light Red → Light Green; Light Green → Dark Green). Those are “market story” drivers.

Output from this chart:

Market state headline (1–2 sentences)

Top flips (10–20 tickers) grouped by flip type

Top “near RevL” lists (Light Red near RevL; Light Green near RevL)

Crowded winners to avoid chasing (long duration + high P/L)

Bleeders to avoid / only monitor (Dark Red long duration, weak MoM)

2) Strategic Universe: Top Momentum Movers (MoM 7–10)

This plot is MoM (x) vs P/L (y), with expanded y-axis 0–50% to reduce crowding. There are momentum bins (10–9.5, 9.5–9, etc.) with clickable ticker chips.

Do this:

Treat it like “leaders board.”

Within each MoM bin:

Identify High MoM + Low/Medium P/L = “fresh leaders / not as extended”

Identify High MoM + Very High P/L = “extended leaders” (needs pullback or tight entries)

Identify MoM high but P/L weak = “momentum improving but not rewarded yet” (possible rotation)

Use R if available: rank the bin by R to find the “true leaders.”

Output:

Top 10 ‘fresh leaders’ (MoM high, P/L < ~35–40%)

Top 10 ‘extended’ (MoM high, P/L very high)

Hidden strength (MoM high, P/L low, improving regime)

Action plan: what to watch for (pullbacks to RevL, reclaim triggers, consolidation breaks)

3) Strategic Universe: Daily Buy Strategic Universe

Scatter shows distribution of names in daily buy mode; sidebar has collapsible bins by “days in buy mode.”

Do this:

Organize names by day-bins:

Day 1–3 = newest buys (fresh flips; highest attention)

Day 4–10 = proving phase

Day 10+ = mature buys; evaluate extension risk

Inside each bin, call out:

Names with low P/L but already in buy mode (early)

Names with very high P/L and long days (crowded)

Identify themes: if many new buys cluster in one industry, call it out.

Output:

Fresh flips (Day 1–3) top list

Best “proving” buys (Day 4–10) (ideally with rising MoM)

Overextended buys (Day 10+ and high P/L)

4) Strategic Universe: Switch Universe (Daily Dark Green + Weekly Light Red)

This is the key “tension” setup: weekly still bearish but daily strong (often early reversal or bear-market rally leaders).

Rules:

Daily must be Dark Green

Weekly must be Light Red

X-axis: weekly weeks in bear trend (TR)

Y-axis: daily P/L (expanded 0–50%)

Do this:

Prioritize names where:

Weekly bear duration is not extreme (early recovery) OR

Weekly bear duration is long but daily strength is accelerating (possible “major turn”)

Distinguish:

Countertrend pops (weekly bear long + daily P/L already huge) vs

Early recovery (weekly bear moderate + daily P/L modest)

Output:

Top 15 Switch candidates split into:

“Early recovery” (best watchlist)

“Late countertrend / crowded bounce” (monitor only)
