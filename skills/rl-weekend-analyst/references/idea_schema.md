## Segment Tag Vocabulary

Use these exact tags for `segments_present`:

### Core Universe
- `regime_flip_dr_to_lr` — Flipped from DR to LR this week
- `regime_flip_lr_to_dg` — Confirmed bullish from reversal attempt
- `revl_proximity_lr` — LR mode within 3% of RevL (reclaim candidate)
- `revl_proximity_lg` — LG mode within 3% of RevL (support test)
- `extended_winner` — DG >20 days, P/L >25%
- `capitulation_inflecting` — DR >30 days, P/L <-15%, MoM turning positive

### Switch Universe
- `switch_fresh_tension` — <10 weeks bear, MoM >3
- `switch_proving_value` — 10-20 weeks bear, P/L <20%
- `switch_proving_extended` — 10-20 weeks bear, P/L ≥20%
- `switch_late_recovery` — >20 weeks bear, MoM >2
- `switch_ultra_long_cluster` — >40 weeks, part of 3+ name cluster
- `switch_positive_r` — In Switch universe with R >0

### Top Momentum
- `mom_fresh_leader` — MoM >8, P/L <15%
- `mom_sweet_spot` — MoM 8.5-9.5, P/L 15-35%
- `mom_extended_leader` — MoM >9, P/L >40%
- `mom_fading` — MoM 7-8

### Daily Buy Mode
- `daily_buy_fresh` — Day 1-3
- `daily_buy_proving` — Day 4-10
- `daily_buy_established` — Day 10-20
- `daily_buy_extended` — Day 20+
- `spbuy_signal` — spBuy or better signal active

### R/Momentum
- `triple_crown` — R>0, MoM>7, days<15
- `r_inflection_positive` — R crossed from negative to positive this week
- `r_high_mom_fading` — R>0 but MoM declining
- `r_negative_mom_rising` — R<0 but MoM >5

### Pullback-in-Strength
- `pullback_shallow` — R>0.5, daily LR, weekly DG, MoM<0 (mildest dip, closest to resolving)
- `pullback_deep` — R>0.5, daily DR, weekly LG, MoM<0 (deeper dip, needs MoM inflection)
- `pullback_max_dislocation` — R>0.5, daily DR, weekly DG, MoM<0 (maximum tension between daily/weekly)
- `pullback_with_spbuy` — Any pullback-in-strength sub-type + active spBuy signal (structural quality + signal confirmation)
- `pullback_sector_cluster` — 3+ names from same sector in pullback-in-strength simultaneously (sector-level dip)

## Confidence Tier Rules

### HIGH (green light)
ALL of the following must be true:
- Probability estimate ≥ 0.65
- Convergence score ≥ 4
- Fundamental alignment confirmed (not contradicted)
- EV > 1.0R
- At least one concrete catalyst within timeframe

### MEDIUM (yellow light)
ANY of the following:
- Probability 0.55-0.65
- Convergence score 3
- Fundamental picture mixed or unclear
- EV 0.5-1.0R
- Catalyst present but timing uncertain

### WATCH (red light — don't trade, just track)
ANY of the following:
- Probability < 0.55
- Convergence score < 3
- Fundamental picture contradicts technical
- EV < 0.5R
- No identifiable catalyst
- Single-segment name with no cross-confirmation

## ID Format

`{date}-{TICKER}-{L|S}` where L=Long, S=Short

Example: `2026-03-14-DKNG-L`

If the same ticker generates ideas on multiple dates, both persist. The system tracks all ideas independently.
